## Research Methodology

When investigating options, evaluating technologies, or forming recommendations:

- Run multiple parallel research threads to collect data from different angles — do not rely on a single source or perspective.
- Every factual claim must be backed by concrete evidence: code references, documentation links, benchmark data, or tool output.
- Distinguish between verified facts and assumptions. Label assumptions explicitly.
- When comparing options, present a structured comparison with consistent criteria across all options.
- Synthesize findings into a clear recommendation with tradeoffs, not just a data dump.
- Cite sources — link to specific documentation, code files, or URLs rather than making unsourced assertions.
